# 基于青稞RISC-V的嵌入式模块化移动终端

旨在设计一台模块化的个人移动终端，以满足用户对移动设备的多样化需求，同时提供灵活的扩展性和定制化能力。

## 模块前期设计

主要包含核心底板、供电模块、显示模块、配件模块、键盘模块、扬声模块

### 核心底板

13寸，主要承载连接所有其他模块、有2A1C1SD接口，提供核心计算能力。

选用CH32V307VCT6，选用128K SRAM,192K FLASH。
板载CH585用于蓝牙、WIFI连接管理。
板载CH378用于tf卡文件系统管理。
板载PCM1794a、OPA1612、功放用于扬声器播放。

一个A口都用于HID设备连接，可以再外接键盘、鼠标。
一个C口用于debug信息输出。

一个A口用于U盘读取。
SD卡接口用于外接存储，采用FAT32文件系统。

有两个槽，一个供电模块槽，一个显示模块槽。
有五个磁吸位置，一个安装显示模块，三个安装配件拓展模块，一个安装键盘模块。
设计开机键、“+”“-”键、Enter键。

### 显示模块

安装在显示磁吸位置，7寸，用于显示，有三种，TFT屏、墨水屏、反射屏。
使用单独的CH32V307QFN68作为显示芯片，核心底板传输压缩过的显示命令给显示模块，然后显示芯片解析为UI布局并渲染。可能采用LVGL，也可能自己写函数类型驱动库，来减少存储容量。三个屏幕的驱动编写有一定程度的不同。
可以上翻60度，改善放在桌面时的观看体验。

### 供电模块

安装在供电槽，可以给核心底板供电，可以拔下来作为一个充电宝使用，提供C口可以输入输出支持快充，提供A口支持慢充。
支持无线充电。

### 键盘模块

安装在键盘磁吸位置，可以有一定程度上翘，预计尺寸270*100,可以选择矮轴/其他。

有多种设计，可以有一些特殊应用的键盘，除了40配列键盘，还可以手柄布局键盘、电子琴布局键盘。

预计使用CH32V103，并且引出一个HID口，用CH9329串口转HID。

### 配件模块

主要满足不同用户的不同需求。
安装在配件磁吸位置，每个模块都需要单独的芯片在模块里面驱动模块，可以选用CH32V103，模块大小不一。
主要包括NFC、旋钮、指纹、触摸圆环、RGB点阵、红外测距、健康监测。

- NFC模块用于读取NFC卡片，可以用于解锁或者读取数据
- 旋钮模块可以执行旋钮操作，需要软件适配
- 指纹模块，用于识别当前用户
- 触摸圆环，类似于旋钮，但是是触摸的而不是拧的
- RGB点阵，可以显示一些小东西，比如天气或者小动画
- 红外测距，可以用于向前的测距
- 健康检测，可以测试当前心率、呼吸、温湿度等
- 副屏模块，显示一些小东西

## 通讯结构

模块间采用UART通讯，以8位数为基本单位,不校验，停止位1,与显示屏和CH378通讯选用460800bps（可能需要支持视频播放,不过解码是个大问题，方案比较复杂），键盘和配件选用115200bps，HID、Debug均选用115200bps。

| Head | Device | Type | Length | Data   | Tail |
| ---- | ------ | ---- | ------ | ------ | ---- |
| 1    | 1      | 1    | 2      | length | 1    |

主要用于核心模块和配件模块、显示模块、键盘模块之间的通讯。

规定Head使用0x45。

Device第一位标识种类，显示使用0x1，键盘使用0x2，配件使用0x3，第二位标识大类之下第几种，也就是说每个大类可以有16种。

Type标识命令还是数据，0x00和0x01

Length标识Data的长度

Tail不校验，只是0x45
