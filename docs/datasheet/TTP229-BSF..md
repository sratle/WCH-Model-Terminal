# TTP229-BSF TonTouch™

## 16键/8键触摸按键检测IC

**版本**：V1.1 | **初稿日期**：2009/12/22 | 页码：共16页
---

## 一、总体描述（GENERAL DESCRIPTION）

TTP229-BSF TonTouch™ 是一款专为触摸按键控制设计的**电容式感应IC**，芯片内置触摸传感器稳压器，感应方案稳定性强，可适配多种使用环境。
该芯片可透过非导电介质连接人机交互面板，主要用于替代传统机械开关/按键。这款专用标准产品（ASSP）可独立配置为**8路触摸按键**或**最多16路触摸按键**。

---

## 二、产品特性（FEATURES）

1. 工作电压：**2.4V ~ 5.5V**
2. 内置稳压器
3. 休眠电流（3V供电，休眠模式、采样率8Hz）
   - 16键模式：典型值 **2.5μA**
   - 8键模式：典型值 **2.0μA**
4. 可通过配置选择**8独立按键**或**16独立按键**模式
5. 8键模式可配置**8路独立并行输出**；8键/16键模式均支持**二线串行输出接口**
6. 8路独立输出可配置驱动类型：CMOS/开漏(OD)/集电极开路(OC)，同时可选择高有效/低有效
7. 二线串行接口可配置高有效/低有效
8. 支持**单键触发/多键同时触发**配置
9. 休眠模式提供两种采样率：低速8Hz、高速64Hz
10. 可通过引脚配置**最长按键锁定时长**，典型约80秒
11. 灵敏度可通过外接电容调节，电容取值范围：**1~50pF**
12. 上电后拥有约**0.5秒稳定时间**，此阶段请勿触摸按键，芯片所有功能暂时禁用
13. 具备**环境自动校准功能**：所有按键长时间无触发时，自动重新校准，重校准周期约4.0秒
    **应用领域（APPLICATION）**
    各类消费电子产品、传统机械按键替代场景

## 七、功能详解（FUNCTION DESCRIPTION）

### 1. 灵敏度调节

PCB上电极尺寸、走线寄生电容都会影响触摸灵敏度，芯片提供多种外部调节方式：

#### 1.1 电极尺寸

其他条件不变：**电极面积越大，灵敏度越高**；面积越小，灵敏度越低（需在有效范围内使用）。

#### 1.2 面板厚度

其他条件不变：**覆盖介质面板越薄，灵敏度越高**；面板越厚，灵敏度越低（不可超过最大限值）。

#### 1.3 外接电容调节（核心方式）

电容取值范围：`1pF ≤ CJ0~CJ3、CJWA、CJWB ≤ 50pF`

- 电容容值越大，灵敏度越低；电容悬空（不接）时，灵敏度最高。
- **CJ0~CJ3**：调节**正常工作模式**下按键灵敏度
- **CJWA、CJWB**：调节**休眠唤醒模式**下灵敏度
- 调节顺序建议：先调CJ0~CJ3，再调CJWA/CJWB
  电容与按键分组对应：
- CJ0：K0、K8
- CJ1：K1、K2、K3、K9、K10、K11
- CJ2：K4、K5、K6、K7、K12、K13
- CJ3：K14、K15
- CJWA：A组唤醒灵敏度（TP0~TP7）
- CJWB：B组唤醒灵敏度（TP8~TP15）
  
  ### 2. 按键数量选择（8键 / 16键）
  
  由 **TP2(KYSEL)** 引脚配置：
- 引脚**不接电阻到VSS**（默认）：**8键模式**
- 引脚**串高阻值电阻接VSS**：**16键模式**
  
  ### 3. 输出模式
  
  #### 3.1 模式总规则
1. 16键模式：**仅支持二线串行输出**
2. 8键模式：同时支持「8路独立并行输出」+「二线串行输出」；并行输出仅8键模式可用
   
   #### 3.2 8路并行输出驱动类型（TP0/OPDEN）
- TP0悬空（默认）：**CMOS推挽输出**
- TP0串高阻电阻接VSS：**开漏(OD)输出**
  
  #### 3.3 并行输出极性 / OD/OC选择（TP1/SAHL）
1. **CMOS输出模式下**
   - TP1悬空（默认）：并行输出**高有效**
   - TP1串高阻电阻接VSS：并行输出**低有效**
2. **OD/OC输出模式下**
   - TP1悬空（默认）：**开漏(OD)**，状态：浮空 / 低有效
   - TP1串高阻电阻接VSS：**集电极开路(OC)**，状态：浮空 / 高有效
     
     > 注意：OD/OC模式引脚内置二极管保护，禁止外接不同电压域器件，避免漏电流。
     
     #### 3.4 二线串行输出接口（SDO+SCL）
- SDO：串行数据输出；SCL：串行时钟输入
- 极性由TP1(SAHL)配置：默认低有效，接电阻到VSS变为高有效
- 工作方式：
  1. 主控轮询读取按键数据
  2. 等待芯片输出数据有效(DV)信号后，主控发时钟读取数据
- 超时机制：SCL引脚**2ms无电平跳变**，串行接口自动复位，回到待机状态
  
  #### 3.5 串行接口时序参数
  
  | 参数          | 最小值 | 典型值 | 最大值  | 单位  |
  | ----------- | --- | --- | ---- | --- |
  | DV          | —   | 93  | —    | μs  |
  | Tw          | 10  | —   | —    | μs  |
  | Tout        | —   | 2   | —    | ms  |
  | T_resp(16键) | —   | 32  | —    | ms  |
  | T_resp(8键)  | —   | 16  | —    | ms  |
  | F_SCL       | 1K  | —   | 512K | Hz  |
  | 时序分类：       |     |     |      |     |
1. TP1=1、TP2=0：16键，高有效
2. TP1=0、TP2=0：16键，低有效
3. TP1=1、TP2=1：8键，高有效
4. TP1=0、TP2=1：8键，低有效
   
   ### 4. 按键触发模式（单键 / 多键）
   
   由 **TP3(SKMS1)、TP4(SKMS0)** 组合配置；引脚接电阻到VSS = 逻辑0，悬空 = 逻辑1。
   按键分组：
- 第一组：TP0、TP1、TP2、TP3、TP8、TP9、TP10、TP11
- 第二组：TP4、TP5、TP6、TP7、TP12、TP13、TP14、TP15
  
  | TP3(SKMS1) | TP4(SKMS0) | 工作模式             |
  | ---------- | ---------- | ---------------- |
  | 1（默认）      | 1（默认）      | 整组16键，**全单键触发**  |
  | 1          | 0          | 两组独立，**两组均单键触发** |
  | 0          | 1          | 两组独立，一组单键、二组多键   |
  | 0          | 0          | 整组16键，**全多键触发**  |
  | 补充说明：      |            |                  |
1. 8键模式仅使用 TP0~TP7；
2. 单键模式下多键同时按下时，**按扫描顺序(TP0→TP15)判定优先级**，与按压力度无关。
   
   ### 5. 休眠模式采样率 & 采样时长
   
   #### 5.1 休眠采样率（TP5/WPSCT）
- 引脚悬空（默认）：**8Hz**
- 引脚串高阻电阻接VSS：**64Hz**
  对应唤醒响应：8Hz→125ms；64Hz→15.6ms
  
  #### 5.2 休眠采样时长（TP6/SLWPTM）
- 引脚悬空（默认）：**4.0ms**
- 引脚串高阻电阻接VSS：**2.0ms**
  
  ### 6. 最长按键锁定时长（TP7/SKSRT）
  
  用于防误触发（异物长期覆盖按键）：
- 引脚悬空（默认）：**关闭限时**，按键可一直触发（无限时长）
- 引脚串高阻电阻接VSS：**开启80秒限时**；超时后输出复位，需重新触摸才可触发
  
  > 注意：开启80秒限时+串行输出时，串行接口仅支持**轮询读取**模式。
  
  ### 7. 内置稳压器
  
  电容触摸电路对电源稳定性要求高，芯片集成稳压器，稳定内部供电，杜绝灵敏度异常、误触发。
  
  ### 8. 自动校准功能
1. 上电后：0.5秒初始化校准，此阶段所有功能禁用，禁止触摸按键；
2. 正常工作后：**所有按键4秒无触发，自动重新环境校准**，循环执行，适配温湿度、环境变化。
   
   ### 9. 休眠→工作模式时序
3. 误触发：短暂触摸唤醒后，4秒无有效按键，自动重校准，回到休眠；
4. 有效触发：触摸唤醒→进入按键扫描（工作模式）；按键松开后，约8秒无新触发，回到休眠。
   
   ### 10. 功能配置总表
   
   > 规则：引脚悬空=状态1；引脚串高阻电阻接VSS=状态0

   | 0          | 1         | OD 低有效       | 低有效     |
   | 0          | 0         | OC 高有效       | 高有效     |
   
   #### （2）按键数量（TP2/KYSEL）
   
   | TP2 | 模式       |
   | --- | -------- |
   | 1   | 8键模式（默认） |
   | 0   | 16键模式    |

## 八、应用电路（APPLICATION CIRCUIT）

### 1. 16键模式应用电路

- 供电：VDD 接电源，VSS 接地；VDD-VSS之间并联104(0.1μF)滤波电容C1，走线尽量短。
- 按键电极：TP0~TP15分别外接触摸电极K0~K15。
- 灵敏度电容：CJ0/CJ1/CJ2/CJ3、CJWA/CJWB 按分组外接1~50pF高精度电容。
- 配置电阻：RP2~RP7（推荐820KΩ）按需接至对应配置引脚与VSS之间，实现功能选择。
- 串行接口：SDO、SCL 外接主控MCU。
  
  ### 2. 8键模式应用电路
- 基础电路同16键；仅使用 TP0~TP7 对应按键 K0~K7。
- 并行输出：TPQ0~TPQ7（TP8~TP15引脚）作为8路独立输出。
- 闲置引脚：不使用串行接口时，**SCL引脚必须接VDD/VSS，禁止悬空**。
  
  ### PCB设计注意事项
1. 触摸电极到芯片引脚的走线**越短越好**，走线禁止平行、交叉；
2. 电源必须稳定，电压快速波动会造成灵敏度异常、误触发；
3. 按键覆盖面板**禁止含金属/导电材质**，表面涂层保持一致；
4. 电源滤波电容C1紧邻VDD/VSS引脚布局，走线最短；
5. 灵敏度电容优先选用 **NPO/X7R** 低温度系数电容，推荐NPO；
6. 配置电阻RP0~RP7 标准取值：**820KΩ**；
7. 不用串行接口时，SCL严禁悬空。
