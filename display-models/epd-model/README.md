# 墨水屏模块 (epd-model)

7寸电子墨水屏，低功耗，适合长时间阅读。

## 硬件配置
- **主控**: CH32V307QFN68 或 CH32H417
- **屏幕**: 7寸电子墨水屏
- **外扩存储**: W25Q128 + W9825G6KH

## 驱动方案
- **接口**: SPI + DMA
- **刷新模式**:
  - GC16: 高质量全刷（灰阶16级）
  - DU: 快速刷新（黑白）
  - A2: 动画模式
- **支持**: 部分刷新、波形控制

## 软件架构 (FreeRTOS + LVGL)
与LCD模块相同的任务架构，针对墨水屏特性优化刷新策略。
